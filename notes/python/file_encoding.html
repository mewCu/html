文件的编码和解码

  • str = 'hello world'
    • 编码 str.encode(encoding='UTF-8')
    • 解码 str.decode(encoding='UTF-8')
  • 如果这个文件是GBK编码
    • file = open(file_path,'r',encoding='GBK') //r模式表示只读取,seek直接的间隔是2bits
  • 如果要用UTF-8编码写入文件
    • file = open(file_path,'w',encoding='UTF-8') //w模式代表覆盖写入
    • file.write('aaa')
