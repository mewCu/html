第一次使用和绑定仓库

  • 创建ssh秘钥
    • 打开bash  输入 ssh-keygen -t rsa -C "你注册github时的邮箱"        回车
    • c盘 用户 下有个 .ssh 文件夹 id_rsa.pub公共秘钥 复制秘钥
  • 上传秘钥
    • github用户setting  打开SSH and GPGkeys  点击new SSH key
    • 验证是否成功 ssh -T git@github.com 
  • 设置name和email
    • git config --global user.name "你的名字"
    • git config --global user.email "你的邮箱"
  • 创建远程仓库
    • 名称和本地仓库最好一样
  • 创建本地仓库
    • git init会生成 .git文件夹
  • 连接到远端仓库
    • git remote add origin /*这里填刚刚远端仓库的SSH地址*/
  • 之后可以执行push和pull操作
