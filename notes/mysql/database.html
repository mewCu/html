数据库创建删除和改名

  • 创建
    • create database website_db;
  • 删除
    • drop database website_db;
  • 改名
    • 创建新数据库
      • create database new_db;
    • 导出旧数据库到文件
      • mysql -uroot -p123456 --set-gtid-purged=OFF old_db > D:/old_db.sql
    • 导入数据到新数据库
      • mysql -uroot -p123456 new_db < D:/old_db.sql
